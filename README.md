# 维修记录本

纸厂仪表维修记录安卓应用

## 功能特性

- ✅ 文字输入：记录维修标题、设备名称、维修说明
- ✅ 语音输入：每个输入框都支持语音识别（中文）
- ✅ 拍照上传：调用系统相机拍照
- ✅ 图片选择：从相册选择图片
- ✅ 搜索功能：按标题/内容/设备名搜索
- ✅ 设备筛选：按设备名称分类筛选
- ✅ 本地存储：数据保存在手机本地

## 下载安装

前往 [Releases](../../releases) 页面下载最新版APK文件，安装到安卓手机即可使用。

## 技术栈

- Kotlin + Jetpack Compose
- Material 3 设计
- Room 数据库
- MVVM 架构

## 最低要求

- Android 8.0 (API 26) 及以上
